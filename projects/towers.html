CS 98 Culminating Project


Web Development: Javascript, HTML, CSS, Processing


Working in a team, this was my Senior Culminating experience. This class was designed to be an open-ended way to self-learn web dev. The project's goal evolved into a way to help new computer science students understand algorithms and learn Javascript. Our module focused on recursion, so we decided to demostrate the Towers of Hanoi. On this project I worked on several aspects on the module the processing code, the javascript that allows the code in the console to execute and move the rings.

We later teamed up with Khan Academy and they used our example on their site.
