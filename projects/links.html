Animated Linkages

Development: Matlab


In this assignment you will design some cool linkages, which will be physically manufactured in a later assignment. We’ll be only looking at kinematics and ignore forces, and furthermore, we’ll limit ourselves to 2D planar structures with revolute joints. Provided with this assignment is a fully functional Matlab code for simulating linkages.
