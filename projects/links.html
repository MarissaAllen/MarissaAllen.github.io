Animated Linkages

Development: Matlab


In this assignment, only looking at kinematics and ignoring forces, I designed some cool 2D planar structures with revolute joints. I was provided with fully functional Matlab code for simulating linkages, I adapted it to create these linkages and modified the code to include an oscillating motion.

Peaucellier-Lipkin Linkage

The Peaucellier-Lipkin Linkage "was the first planar linkage capable of transforming rotary motion into perfect straight-line motion, and vice versa.”-Wikipedia


Klann Linkage

For this mechanism, the geometry of some of the links will no longer be rectangles. You’ll need to modify the verts field of link to make this change. Note that the drawScene() function draws the link geometry by connecting the vertices in the order they are defined.
